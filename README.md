## 📌 About

Climate Guard is an AI + AR powered web application that recommends the best indoor and outdoor plants based on your city's **real-time weather conditions** — and then lets you **visualize those plants in your actual room** using Augmented Reality, directly in the browser. No app installation required.

> 📜 Co-filed IP Patent with PSIT Kanpur — Application No. pending (Feb 2026)
> 🏅 Recognized at Tech Expo 2025, PSIT Kanpur by industry experts

---

## ✨ Features

- 🌦️ **Real-time weather detection** — Fetches live temperature, humidity & sunlight data for any city
- 🌱 **Smart plant recommendations** — Suggests suitable indoor & outdoor plants based on climate
- 🖼️ **AI room analysis** — Upload a photo of your empty space; AI places plants accurately inside it
- 🥽 **3D AR overlay** — View plants in your real-world environment using Google Model-Viewer & WebXR
- 📱 **No app needed** — Fully browser-based AR, works on any modern mobile or desktop device

| Layer | Technology |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/kavya-dev05/Climate-Guard.git

# Navigate into the project
cd Climate-Guard

# Open in browser
open index.html
```

> Add your Weather API key and Generative AI API key in the config file before running.
